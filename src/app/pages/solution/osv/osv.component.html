A software profile comprises of inventory and operational preferences, designed to process the software recommendations

  • Inventory Preferences:
    • This comprises all devices in the customer network organised via a group policy.
    • A policy group organises assets (currently bases on Software Type and Product Id).
    • It aggregates the inventory information from all the grouped assets.
  • Operational Preferences:
    • This comprises a set of attributes that define the preferred operational aspects for software recommendation, such as Release Age and Release Popularity.
    • Based on release attributes, there are different categories such as Conservative, Moderate and Aggressive.
    • Currently software profiles are set to Aggressive by default which will prefer latest releases.
